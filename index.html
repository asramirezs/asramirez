Bienvenido a mi sitio Web!


Este es un espacio en el cual podrá interactuar con la página Web, conociendo cada uno de los campos relacionados en el diseño de sitios Web, y demostrando los conocimientos adquiridos en este curso.


Objetivo

Facilitar la posibilidad de que el usuario pueda visualizar de una forma amigable el sistema web


HTML5

Es un lenguaje de etiquetas que permite construir páginas web de forma que los navegadores puedan entender el contenido y mostrárselo al usuario


CSS

El CSS nos permite controlar los estilos de nuestros elementos HTML, como lo son: los colores, fuentes, ubicación y todo lo que se va a mostrar de nuestra web. .
